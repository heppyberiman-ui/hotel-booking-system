# 🏨 Grand Horizon Hotel Booking System

## 📖 Deskripsi Project

Grand Horizon Hotel Booking System adalah aplikasi web berbasis Fullstack yang digunakan untuk melakukan reservasi kamar hotel secara online. Sistem ini memungkinkan pelanggan untuk melihat informasi kamar, melakukan pemesanan, mengunggah bukti pembayaran, serta memantau status reservasi. Selain itu, sistem menyediakan Portal Admin untuk mengelola seluruh operasional hotel.

---

## 🎯 Tujuan Sistem

Menyediakan platform reservasi hotel secara online yang dapat digunakan oleh pelanggan dan admin secara real-time sehingga proses pemesanan menjadi lebih cepat, efisien, dan terkomputerisasi.

## 🔄 Alur Sistem

1. Customer melakukan registrasi atau login.
2. Customer memilih kamar yang tersedia.
3. Customer mengisi tanggal check-in dan check-out.
4. Customer melakukan booking kamar.
5. Customer mengunggah bukti pembayaran.
6. Admin melakukan verifikasi pembayaran.
7. Status booking berubah menjadi Confirmed.
8. Customer melakukan check-in ke hotel.

## 🚀 Cara Menjalankan Project

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```

### Database

1. Jalankan XAMPP.
2. Aktifkan Apache dan MySQL.
3. Import file database SQL ke phpMyAdmin.
4. Sesuaikan konfigurasi database pada file `.env`.
